# Category illustration system

The thirteen category backgrounds were generated with the built-in GPT Image workflow and optimized to 960-pixel WebP files for the showcase.

## Shared art direction

- Refined 1960s Swiss editorial illustration fused with risograph screen printing.
- Flat geometric forms, bold black keylines, imperfect ink registration, and subtle paper grain.
- Landscape, full-bleed compositions that remain legible on small category cards.
- No text, letters, numbers, logos, UI screenshots, gradients, photorealism, watermarks, or borders.
- Every image uses cream and deep espresso as common anchors, then adds a distinct category palette.
